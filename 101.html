详细部署指南

步骤详解：

1. 在GitHub创建新仓库 (Repository) - 点击右上角 "+" 号 → "New repository" - 输入仓库名称，例如 "my-animated-website" - 选择 "Public" (公开) - 勾选 "Add a README file"
2. 上传你的网页文件 - 点击 "Add file" → "Upload files" - 将你的HTML、CSS、JS文件拖拽到上传区域 - 添加提交信息并提交
3. 启用GitHub Pages - 进入仓库 "Settings" 选项卡 - 左侧菜单找到 "Pages" - 在 "Source" 下选择部署分支 (通常是main) - 选择根目录 "/ (root)" - 点击保存
4. 访问你的网站 - 等待几分钟后访问: https://你的用户名.github.io/仓库名称
